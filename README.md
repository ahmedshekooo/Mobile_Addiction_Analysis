# Mobile Addiction Analysis 📱📊

An interactive data analysis web application built with Streamlit that explores, visualizes, and analyzes mobile screen time patterns, usage habits, and indicators of smartphone dependency or digital fatigue. 

Live Dashboard: **[View the App](https://mobileaddictionanalysis-x4qeexz8zs8esz7j8dtyib.streamlit.app/)**

---

## 🚀 Features

- **Interactive Visualizations:** Deep-dive charts showing usage patterns, peak times, and app categories that dominate screen time.
- **Behavioral Analysis:** Analytical insights into how mobile habits affect daily life, productivity, or sleep routines.
- **User Segmentation:** Data-driven categorization of users based on severity metrics or behavioral indicators.
- **Clean User Interface:** Simple, easy-to-use controls leveraging the power of Streamlit for filtering and toggling metrics.

---

## 🛠️ Built With

* **Python** — Core backend and data manipulation
* **Streamlit** — Web application framework for interactive dashboards
* **Pandas / NumPy** — Data processing, cleansing, and aggregation
* **Matplotlib / Seaborn / Plotly** — Interactive and publication-quality visual plots
